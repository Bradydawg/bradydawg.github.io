Hub Navigator

Name Color/Chat Color

I was contracted to build a plugin that allowed users (with the proper permissions) to modify their name color and chat color with the help of an easy to use, and nice looking GUI.

Chat Formatting

I was contracted to build a plugin that formatted the default chat into something that was nice to look at, while hooking into permissions plugins, and nickname plugins, to allow ranks to be displayed, as well as chat color modification, and name modification (nicknames, and name color).

Private Message

I was contracted to build a plugin that allowed users to send each other private messages that only they were allowed to see. A cool feature in this plugin is the socialspy feature, which allows administrators with the proper permissions, to see everyones private messages.
